Secretaria online

O objetivo dessa aplicação é testar as habilidades técnicas dos(as) novos(as) instrutores(as) de desenvolvimento web front-end.

A Generation está buscando Instrutores para facilitar treinamentos para jovens beneficiados pelo Programa Generation - Desenvolvedor Java Jr. O programa busca qualificar e inserir jovens no mercado de trabalho com uma formação rápida no estilo bootcamp.

O Instrutor será responsável por facilitar sessões nas quais os jovens desenvolverão habilidades técnicas e socioemocionais para qualificá-los para vagas em empregadores parceiros da Generation.
